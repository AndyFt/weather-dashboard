## Description

Unleash your inner meteorologist with this JavaScript-powered weather dashboard! Track current conditions and 5-day forecasts for your favorite cities with ease. Search, save, and revisit locations effortlessly, thanks to local storage that remembers your preferences. ## Usage

The user can choose a city to check the weather for today and the forecast for the next 5 days. If you refresh the page, you will find your latest searches. </br>
